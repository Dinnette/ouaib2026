### Des notions de HTML

1. HTML
HyperText Markup Language (langage de balise hypertexte)

=> un fichier `.HTML`, c'est d'abord du texte.
=> il est écrit par un `développeur` (avec VScode par ex)
=> il est vu par un utilisateur (avec firefox par ex)


Une balise `html` s'écrit : `<balise></balise>`
